# ASE WAV Website

This is a static website for `asewav.com`. It can be hosted for free on Cloudflare Pages, GitHub Pages, or a similar static host.

## What to update later

- Replace the "Notify me when it drops" button in `index.html` with your Payhip checkout link when the ebook page is ready.
- If you create a separate MailerLite form or group for the free sample, replace the form `action` URL in the `#sample` section.
- The current hero and preview images are stored in `assets/` as web-sized JPEGs.

## Suggested hosting path

Use Cloudflare Pages and connect the custom domain `asewav.com`. Upload or deploy this folder as the site root.
